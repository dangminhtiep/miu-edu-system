# START NEW SESSION PROMPT - MIU WEB

Last updated: 2026-03-06
Purpose: short prompt template the user can paste at the start of a fresh chat session

## Recommended Prompt
```text
Day la phien moi cho repo `miu-web`.

Truoc khi lam viec, hay doc lai bo nho ngoai cua du an theo dung quy trinh:
- PROJECT_OVERVIEW.md
- PROJECT_MEMORY.md
- DECISIONS_LOG.md
- MODULE_STATUS.md
- ENGINEERING_STANDARDS.md
- OPEN_QUESTIONS.md
- RISK_REGISTER.md
- SESSION_HANDOFF.md
- DOC_MAINTENANCE_PROTOCOL.md
- COLLABORATION_CONVENTIONS.md
- WHEN_TO_START_NEW_SESSION.md

Sau khi doc xong:
1. tom tat lai cho toi:
- chung ta da chot nhung gi
- dang o giai doan nao
- viec tiep theo nen lam la gi
2. chi ro neu co bat ky diem nao mo hoac co nguy co lech ngu canh
3. trong phien nay, neu co quyet dinh moi thi cap nhat docs dung file theo protocol truoc khi ket thuc phien

Lưu ý trao đổi với tôi bằng Tiếng Việt có dấu
```

## Usage Rule
- Paste this at the start of a new session whenever continuity matters.
- If the previous session ended after a major checkpoint, this prompt should be enough to resume safely.
